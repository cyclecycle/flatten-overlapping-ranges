# flatten-overlapping-ranges

**Flatten overlapping ranges into a sequence of sections.**

```
---A---     ---C---
  -----B------
     --D--
```

```
-- --- -- --- -- -- -----
A  A   A         C  C
   B   B  B   B  B
       D  D
```
